# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of Thank You Stack Overflow
- User profile data fetching from Stack Overflow API
- Gratitude card generation with user statistics
- Image export functionality using html2canvas
- Twitter/X sharing integration
- Mobile device support with app deep linking
- Option to share gratitude without Stack Overflow account
- Beautiful modal UI for sharing instructions
- Responsive design for all screen sizes

### Features
- Display user reputation, badges, questions, and answers
- Show top upvoted question and answer
- Member since date
- Profile image display
- Custom gratitude message generation
